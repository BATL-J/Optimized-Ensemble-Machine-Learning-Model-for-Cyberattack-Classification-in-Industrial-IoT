# Optimized Ensemble Machine Learning Model for-Cyberattack Classification in Industrial IoT
This repository contains the code and datasets associated with the research paper titled "Optimized Ensemble Machine Learning Model for Cyberattack Classification in Industrial IoT" by Batool Alabdullah and Prof.Suresh Sankaranarayanan.

## Requirements & Python Nootbook
- **Requirements :**
  - Install the dependencies in the `Requirements.txt` file, to run the experiments
